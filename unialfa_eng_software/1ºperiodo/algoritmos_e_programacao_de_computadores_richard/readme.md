# 🧠 Algoritmos e Programação de Computadores

Repositório dedicado aos conteúdos, exercícios e estudos desenvolvidos durante a disciplina de **Algoritmos e Programação de Computadores**.

O foco desta matéria é fortalecer a base lógica da programação, praticando resolução de problemas, algoritmos e estruturas fundamentais utilizando linguagem C.

---

# 📂 Estruturas Básicas de Dados

Diretório voltado para estudo e futura implementação prática das principais estruturas de dados.

## 📚 Estruturas estudadas

### 📦 Pilha
Estrutura baseada no conceito LIFO *(Last In, First Out)*.

### 📥 Fila
Estrutura baseada no conceito FIFO *(First In, First Out)*.

### 📋 Lista
Estrutura utilizada para armazenamento sequencial de dados.

---

## 🚀 Objetivos

- Entender o funcionamento interno das estruturas
- Desenvolver implementações próprias em C
- Praticar manipulação de memória
- Fortalecer lógica de programação
- Aprender organização de algoritmos
