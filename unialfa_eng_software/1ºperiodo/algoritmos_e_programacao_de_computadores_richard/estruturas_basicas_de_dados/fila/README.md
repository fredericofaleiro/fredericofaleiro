# 📥 Fila

Diretório dedicado ao estudo da estrutura de dados **Fila** utilizando linguagem C.

A fila funciona seguindo o conceito **FIFO** *(First In, First Out)*, onde o primeiro elemento inserido é o primeiro a ser removido.

---

# 📚 Conceitos estudados

- Inserção no final da fila
- Remoção no início da fila
- Controle de entrada e saída
- Organização sequencial de dados

---

# 🧠 Aplicações da Fila

- Sistemas de atendimento
- Filas de impressão
- Processamento de tarefas
- Gerenciamento de processos
- Controle de requisições

---

# 🚀 Futuras implementações

- Fila estática
- Fila dinâmica
- Fila circular
- Manipulação utilizando ponteiros

---

# 🛠️ Tecnologias utilizadas

- Linguagem C
- Markdown

---

# 🎯 Objetivo

Este diretório tem como objetivo estudar e implementar o funcionamento de filas, compreendendo sua lógica, aplicações e manipulação de dados em memória.
