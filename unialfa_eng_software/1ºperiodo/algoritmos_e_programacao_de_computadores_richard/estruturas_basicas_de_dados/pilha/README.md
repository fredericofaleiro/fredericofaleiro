# 📦 Pilha

Diretório dedicado ao estudo da estrutura de dados **Pilha** utilizando linguagem C.

A pilha funciona seguindo o conceito **LIFO** *(Last In, First Out)*, onde o último elemento inserido é o primeiro a ser removido.

---

# 📚 Conceitos estudados

- Empilhar elementos *(push)*
- Desempilhar elementos *(pop)*
- Consulta ao topo da pilha
- Controle de fluxo
- Organização sequencial de dados

---

# 🧠 Aplicações da Pilha

- Histórico de navegação
- Desfazer e refazer ações
- Chamadas de funções
- Processamento de expressões matemáticas
- Controle de execução de programas

---

# 🚀 Futuras implementações

- Pilha estática
- Pilha dinâmica
- Manipulação utilizando ponteiros
- Controle de memória em C

---

# 🛠️ Tecnologias utilizadas

- Linguagem C
- Markdown

---

# 🎯 Objetivo

Este diretório tem como objetivo estudar e implementar o funcionamento interno de pilhas, fortalecendo conceitos de lógica, memória e estruturas de dados.
