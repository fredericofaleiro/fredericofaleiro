# 🏋️ FITB

Projeto desenvolvido durante a disciplina de **Oficina de Programação**, com foco na criação de uma landing page moderna para uma academia fictícia.

O projeto foi construído utilizando HTML5 e CSS3, explorando conceitos de responsividade, organização de layouts e estruturação de páginas web.

---

# 🚀 Funcionalidades

- Página inicial responsiva
- Menu de navegação mobile
- Seção de aulas da academia
- Página com planos
- Página de cadastro de membros
- Cards interativos
- Layout adaptado para diferentes tamanhos de tela
- Área de contato
- Redes sociais

---
