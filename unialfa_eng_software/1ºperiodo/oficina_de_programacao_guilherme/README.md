# 💻 Oficina de Programação

Repositório dedicado aos conteúdos e projetos desenvolvidos durante a disciplina de **Oficina de Programação**.

A disciplina tem foco no desenvolvimento web utilizando HTML, CSS e JavaScript, abordando desde a estruturação de páginas até manipulação dinâmica do HTML com JavaScript.

---

# 📂 Projetos

## 🏋️ FITB

Projeto de uma página de academia fictícia desenvolvido utilizando HTML e CSS.

### 🚀 Objetivos do projeto

- Estruturação de páginas web
- Estilização com CSS
- Organização visual
- Responsividade
- Prática de layout
