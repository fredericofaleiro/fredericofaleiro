# 🧠 Estruturas Básicas de Dados

Diretório dedicado ao estudo teórico e futuro desenvolvimento prático das principais estruturas de dados utilizando linguagem C.

Atualmente cada estrutura possui documentação em Markdown explicando seu funcionamento, objetivos e aplicações.

---

# 📂 Estruturas disponíveis

## 📦 Pilha

Estrutura baseada no conceito **LIFO** *(Last In, First Out)*, onde o último elemento inserido é o primeiro a ser removido.

### 📌 Conceitos estudados
- Empilhar elementos
- Desempilhar elementos
- Controle de topo
- Organização sequencial

---

## 📥 Fila

Estrutura baseada no conceito **FIFO** *(First In, First Out)*, onde o primeiro elemento inserido é o primeiro a sair.

### 📌 Conceitos estudados
- Inserção no final
- Remoção no início
- Controle de fluxo
- Organização sequencial

---

## 📋 Lista

Estrutura utilizada para armazenar elementos de maneira organizada e sequencial.

### 📌 Conceitos estudados
- Inserção de elementos
- Remoção de elementos
- Percorrer listas
- Organização dinâmica de dados
