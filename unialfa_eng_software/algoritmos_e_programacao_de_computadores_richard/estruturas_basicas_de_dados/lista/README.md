# 📋 Lista

Diretório dedicado ao estudo da estrutura de dados **Lista** utilizando linguagem C.

As listas permitem armazenar elementos de forma organizada e sequencial, sendo uma das estruturas fundamentais da programação.

---

# 📚 Conceitos estudados

- Inserção de elementos
- Remoção de elementos
- Percorrer listas
- Organização dinâmica de dados
- Manipulação de estruturas

---

# 🧠 Aplicações das Listas

- Armazenamento dinâmico de dados
- Organização de coleções
- Estruturas encadeadas
- Manipulação de registros
- Base para estruturas mais avançadas

---

# 🚀 Futuras implementações

- Lista encadeada
- Lista duplamente encadeada
- Lista circular
- Manipulação utilizando ponteiros

---

# 🛠️ Tecnologias utilizadas

- Linguagem C
- Markdown

---

# 🎯 Objetivo

Este diretório tem como objetivo estudar e implementar listas em linguagem C, fortalecendo conhecimentos em estruturas de dados, memória e lógica de programação.
