# 🎟️ Sistema de Ingressos

Projeto desenvolvido durante os estudos de lógica de programação na Alura.

A aplicação simula um sistema de compra de ingressos com controle de quantidade disponível em diferentes setores.

---

# 🚀 Funcionalidades

- Comprar ingressos por setor
- Controle de estoque em tempo real
- Validação de quantidade disponível
- Atualização dinâmica da interface
- Alertas para ingressos esgotados

---

# 🧠 Conceitos praticados

- Condicionais
- Manipulação de DOM
- Conversão de valores
- Eventos
- Controle de fluxo
- Atualização dinâmica de conteúdo

---

# 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

# 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Controle de dados utilizando JavaScript
- Manipulação dinâmica de elementos HTML
- Estruturação de lógica condicional
- Validação de entradas do usuário

---

Este projeto foi desenvolvido para fortalecer meus conhecimentos em lógica de programação e interação com interfaces web.
