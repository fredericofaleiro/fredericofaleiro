# 👥 Amigo Secreto

Projeto desenvolvido durante os estudos de lógica de programação na Alura.

A aplicação realiza o sorteio automático de amigo secreto entre participantes cadastrados pelo usuário.

---

# 🚀 Funcionalidades

- Adicionar participantes
- Validar nomes duplicados
- Validar campos vazios
- Sorteio automático
- Embaralhamento aleatório dos participantes
- Reiniciar sorteio

---

# 🧠 Conceitos praticados

- Arrays
- Loops
- Validação de dados
- Manipulação de DOM
- Algoritmo de embaralhamento
- Eventos
- Condicionais

---

# 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

# 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Manipulação de listas em JavaScript
- Atualização dinâmica da interface
- Controle de fluxo com condicionais
- Algoritmos de randomização
- Interação entre usuário e aplicação

---

# 🎯 Objetivo

Este projeto foi desenvolvido para fortalecer meus conhecimentos em lógica de programação e manipulação de elementos HTML com JavaScript.
