# 🔢 Jogo do Número Secreto

Projeto desenvolvido durante os estudos de lógica de programação na Alura.

A aplicação consiste em um jogo interativo onde o usuário deve descobrir um número aleatório gerado pelo sistema.

---

# 🚀 Funcionalidades

- Geração aleatória de números
- Controle de tentativas
- Dicas de maior ou menor
- Reinício automático do jogo
- Sistema de voz utilizando Speech Synthesis API
- Controle de números já sorteados

---

# 🧠 Conceitos praticados

- Funções
- Recursão
- Arrays
- Condicionais
- Loops
- Randomização
- Manipulação de DOM
- Speech Synthesis API

---

# 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

# 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Estruturação de lógica de jogo
- Manipulação dinâmica de interface
- Controle de estado da aplicação
- Algoritmos de randomização
- Uso de APIs do navegador

---

# 🎯 Objetivo

Este projeto foi desenvolvido para consolidar fundamentos de lógica de programação e desenvolvimento web utilizando JavaScript.
