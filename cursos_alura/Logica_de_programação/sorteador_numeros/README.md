# 🎲 Sorteador de Números

Projeto desenvolvido durante os estudos de lógica de programação na Alura.

A aplicação realiza sorteios automáticos de números aleatórios sem repetição dentro de um intervalo definido pelo usuário.

---

# 🚀 Funcionalidades

- Definir quantidade de números
- Escolher intervalo de sorteio
- Evitar números repetidos
- Reiniciar sorteio
- Atualização dinâmica da interface
- Validação de entradas

---
