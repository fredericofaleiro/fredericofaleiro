# 🎮 AluGames

Projeto desenvolvido durante os estudos de lógica de programação na Alura.

O objetivo da aplicação é simular um sistema simples de aluguel de jogos, permitindo alterar dinamicamente o status entre disponível e alugado.

---

# 🚀 Funcionalidades

- Alterar status do jogo entre alugado e disponível
- Mudança dinâmica de botão e imagem
- Confirmação antes de devolver um jogo
- Atualização visual em tempo real
- Registro de ações no console

---

# 🧠 Conceitos praticados

- Manipulação de DOM
- querySelector
- getElementById
- Eventos
- Condicionais
- Manipulação de classes CSS
- Template Strings
- Organização de funções
- Interatividade com JavaScript

---

# 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

# 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Seleção e manipulação de elementos HTML
- Alteração dinâmica de classes CSS
- Criação de interações com usuário
- Estruturação de lógica utilizando funções
- Atualização de interface em tempo real

---

# 🎯 Objetivo

Este projeto faz parte da minha evolução nos estudos de desenvolvimento web e lógica de programação, servindo como prática para manipulação de interface e construção de aplicações interativas com JavaScript.
