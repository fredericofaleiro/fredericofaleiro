# 🛒 Carrinho de Compras

Projeto desenvolvido durante os estudos de lógica de programação na Alura.

A aplicação simula um carrinho de compras com cálculo automático de subtotal e valor total dos produtos adicionados.

---

# 🚀 Funcionalidades

- Adicionar produtos ao carrinho
- Calcular subtotal automaticamente
- Atualizar valor total em tempo real
- Limpar carrinho
- Exibir produtos adicionados dinamicamente

---

# 🧠 Conceitos praticados

- Arrays
- Manipulação de DOM
- Operações matemáticas
- Eventos
- Template Strings
- Conversão de valores
- Atualização dinâmica de interface

---

# 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
