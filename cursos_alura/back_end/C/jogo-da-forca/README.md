# 🔠 Jogo da Forca

Projeto de jogo da forca desenvolvido em linguagem C com sistema de palavras carregadas por arquivo e possibilidade de adicionar novas palavras ao banco de dados.

---

# 🚀 Funcionalidades

- Sistema completo de forca
- Leitura de palavras por arquivo `.txt`
- Verificação de letras corretas e incorretas
- Desenho da forca em ASCII
- Sistema de vitória e derrota
- Cadastro de novas palavras
- Controle de tentativas

---

# 🛠️ Tecnologias utilizadas

- Linguagem C
- `stdio.h`
- `stdlib.h`
- `string.h`
- `time.h`

---

# 📚 Conceitos praticados

- Strings
- Vetores
- Manipulação de arquivos
- Modularização
- Estruturas condicionais
- Loops
- Funções
- Organização de código

---

# 🎯 Objetivo do projeto

Projeto desenvolvido para aprofundar conhecimentos em manipulação de arquivos, modularização e lógica de programação utilizando linguagem C.
