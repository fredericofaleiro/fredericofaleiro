# 💻 Linguagem C — Projetos e Estudos

Este diretório reúne meus projetos desenvolvidos em **linguagem C**, focados principalmente em lógica de programação, estruturas fundamentais, manipulação de memória, arquivos e desenvolvimento de jogos no terminal.

Os projetos foram criados durante meus estudos na Alura e servem como prática para fortalecer minha base em programação de baixo nível e raciocínio lógico.

---

# 📂 Projetos

## 🎯 Adivinhar Número

Jogo de adivinhação desenvolvido no terminal onde o jogador precisa descobrir um número secreto gerado aleatoriamente.

### 🚀 Funcionalidades
- Sistema de dificuldade
- Número aleatório
- Sistema de pontuação
- Controle de tentativas
- Feedback se o número é maior ou menor
- Tratamento de entradas inválidas

### 🛠️ Tecnologias
- Linguagem C
- Biblioteca `stdio.h`
- Biblioteca `stdlib.h`
- Biblioteca `time.h`
- Biblioteca `locale.h`

### 📚 Conceitos praticados
- Variáveis
- Loops
- Condicionais
- Funções da biblioteca padrão
- Geração de números aleatórios
- Estruturas de repetição
- Controle de fluxo

## 🔠 Jogo da Forca

Projeto completo de jogo da forca no terminal com leitura de palavras por arquivo e possibilidade de adicionar novas palavras dinamicamente.

### 🚀 Funcionalidades
- Sistema de tentativas
- Desenho da forca em ASCII
- Leitura de palavras em arquivo `.txt`
- Verificação de letras repetidas
- Sistema de vitória e derrota
- Cadastro de novas palavras

### 🛠️ Tecnologias
- Linguagem C
- Manipulação de arquivos
- Strings
- Modularização com `.h`
- Biblioteca `string.h`

### 📚 Conceitos praticados
- Vetores
- Strings
- Modularização
- Funções
- Manipulação de arquivos
- Estruturas condicionais
- Loops
- Organização de código
