# 🎯 Adivinhar Número

Jogo de adivinhação desenvolvido em linguagem C no terminal, onde o jogador deve descobrir um número secreto gerado aleatoriamente pelo sistema.

---

# 🚀 Funcionalidades

- Sistema de dificuldade
- Número aleatório
- Controle de tentativas
- Sistema de pontuação
- Feedback se o chute é maior ou menor
- Tratamento de números inválidos
- Interface em ASCII no terminal

---

# 🛠️ Tecnologias utilizadas

- Linguagem C
- `stdio.h`
- `stdlib.h`
- `time.h`
- `locale.h`

---

# 📚 Conceitos praticados

- Variáveis
- Loops
- Condicionais
- Funções
- Geração de números aleatórios
- Controle de fluxo
- Entrada e saída de dados

---

# 🎯 Objetivo do projeto

Projeto desenvolvido para praticar fundamentos da programação em C, lógica de programação e estruturação de aplicações no terminal.
