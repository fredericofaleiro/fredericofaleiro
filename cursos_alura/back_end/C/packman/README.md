# 👾 Packman

Projeto inspirado no clássico Pac-Man desenvolvido em linguagem C utilizando mapas carregados por arquivo.

> ⚠️ Projeto em desenvolvimento.

---

# 🚀 Funcionalidades atuais

- Leitura de mapa por arquivo
- Movimentação do personagem
- Sistema de colisão
- Impressão dinâmica do mapa
- Controle via teclado
- Estrutura modularizada

---

# 🔧 Próximas implementações

- Fantasmas
- Sistema de pontuação
- Vitória e derrota
- Itens coletáveis
- Melhorias visuais

---

# 🛠️ Tecnologias utilizadas

- Linguagem C
- Ponteiros
- Alocação dinâmica de memória
- Manipulação de arquivos
- Structs
- Modularização com `.h`

---

# 📚 Conceitos praticados

- Matrizes

- Ponteiros
- Memória dinâmica
- Organização de projetos
- Manipulação de mapas

# 🎯 Objetivo do projeto

Projeto desenvolvido para praticar conceitos mais avançados da linguagem C, incluindo manipulação de memória, modularização e desenvolvimento de jogos no terminal.
