# ⚙️ Back-end

Repositório dedicado aos estudos de back-end durante minha jornada de aprendizado na Alura.

Aqui concentro projetos voltados para lógica de programação, estruturação de aplicações e desenvolvimento utilizando linguagens como C e Java.

---

# 📂 Estrutura do repositório

## 🔹 C

Projetos desenvolvidos para prática de lógica, algoritmos e programação estruturada.

### 🎯 Projetos

#### 🔢 Adivinhar Número
Jogo simples onde o usuário tenta descobrir um número gerado aleatoriamente.

#### 🎮 Jogo da Forca
Sistema completo de jogo da forca utilizando manipulação de arquivos, funções e lógica de jogo.

#### 👾 Packman
Projeto focado em movimentação, lógica de jogo e manipulação de estruturas em C.
