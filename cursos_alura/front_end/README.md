# 📚 Cursos Alura — Front-end

Repositório criado para armazenar estudos, exercícios e projetos desenvolvidos durante os cursos de Front-end realizados na Alura.

Aqui organizo conteúdos voltados para desenvolvimento web, lógica de programação e criação de aplicações utilizando HTML, CSS, JavaScript e React.

---

# 👨‍💻 Sobre o repositório

Este repositório reúne exercícios, testes práticos e projetos desenvolvidos ao longo dos cursos da Alura, com foco no aprimoramento das habilidades em desenvolvimento Front-end.

Os conteúdos vão desde conceitos básicos da web até aplicações mais completas utilizando React, LocalStorage e integração com banco de dados.

# 🚀 Conteúdos estudados

## 🧠 JavaScript

Estudos focados em lógica de programação, manipulação de dados e interatividade para aplicações web.

### 📂 Exercícios simples

Pasta utilizada para testes e práticas de conceitos fundamentais do JavaScript, incluindo:

* Arrays
* Objects
* Funções
* Loops
* Condicionais
* Manipulação de dados
* Testes de lógica

### 🚀 Projetos

#### ⏱️ Fokus

Aplicação desenvolvida utilizando JavaScript com persistência de dados através do LocalStorage.

### 📚 Conceitos praticados

* Manipulação do DOM
* Eventos
* Temporizadores
* Armazenamento local
* Organização de código

---

#### 🗂️ Memoteca

Projeto mais completo desenvolvido com integração entre Front-end e banco de dados.

### 📚 Recursos utilizados

* Servidor local
* Banco de dados
* Requisições HTTP
* CRUD
* Organização de aplicações
* Estruturação de componentes

---

## 🌐 HTML & CSS

Projetos voltados para estruturação e estilização de páginas web.

### 📂 Projetos desenvolvidos

#### ✈️ Jornada Viagens

Projeto focado em criação de layouts modernos e organização visual.

#### 🎨 Pesquisa Culturama

Projeto desenvolvido para prática de responsividade e estruturação de páginas.

#### 💻 TechBoard

Interface desenvolvida utilizando HTML e CSS, servindo como base visual para a futura versão em React.

### 📚 Conceitos praticados

* HTML5
* CSS3
* Flexbox
* Grid Layout
* Responsividade
* Organização visual
* Estruturação de páginas

---

## ⚛️ React

Estudos focados na criação de interfaces modernas e componentização utilizando React.

### 💻 TechBoard React

Versão mais funcional do projeto TechBoard desenvolvida utilizando React.

### 📚 Conceitos praticados

* Componentização
* Props
* Estados
* Hooks
* Reutilização de componentes
* Organização de aplicações
* Interfaces dinâmicas
