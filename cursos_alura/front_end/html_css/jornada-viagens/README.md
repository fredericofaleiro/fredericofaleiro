# ✈️ Jornada Viagens

Projeto desenvolvido utilizando HTML5 e CSS3 com foco na criação de uma interface moderna, organizada e responsiva.

Este é um dos projetos mais completos desenvolvidos durante meus estudos de Front-end, reunindo conceitos importantes de estruturação visual e organização de código.

---

# 🎯 Objetivo do projeto

Criar uma página moderna para uma plataforma fictícia de viagens, aplicando boas práticas de organização visual e desenvolvimento Front-end.
