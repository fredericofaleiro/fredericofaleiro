# 🎨 Pesquisa Culturama

Projeto desenvolvido utilizando HTML5 e CSS3 com foco na criação de um formulário moderno, organizado e responsivo.

A aplicação simula uma pesquisa cadastral, contendo diversos campos para preenchimento de informações do usuário.

---

# 🎯 Objetivo do projeto

Praticar estruturação de formulários utilizando HTML e estilização com CSS, aplicando conceitos de organização visual e responsividade.

---

# 📚 Conceitos praticados

* HTML5
* CSS3
* Formulários
* Inputs
* Labels
* Organização visual
* Responsividade
* Estruturação de páginas
