# 🚀 Projetos em destaque

## ⏳ Fokus

Aplicação inspirada na técnica Pomodoro para aumentar produtividade e foco.

### Funcionalidades

- Temporizador dinâmico
- Modos de foco e descanso
- Música ambiente
- Gerenciamento de tarefas
- Uso de LocalStorage

### Tecnologias

- HTML
- CSS
- JavaScript

---

## 🧠 Memoteca

Aplicação CRUD desenvolvida para armazenar pensamentos, frases e ideias.

### Funcionalidades

- Criar pensamentos
- Editar pensamentos
- Excluir pensamentos
- Integração com API fake usando JSON Server
- Requisições HTTP (GET, POST, PUT e DELETE)

### Tecnologias

- HTML
- CSS
- JavaScript
- JSON Server
- API REST
